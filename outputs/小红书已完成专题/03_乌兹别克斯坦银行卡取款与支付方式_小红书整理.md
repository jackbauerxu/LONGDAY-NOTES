## 一、核心结论

1. 现金仍然是乌兹别克斯坦旅行最稳妥的支付方式。
   打车、市场、小店、部分交通和临时场景都建议准备现金索姆。

2. 银联卡 ATM 取现可行，而且多篇帖子实测成功。
   塔什干、撒马尔罕、塔什干火车站北站等地点都有银联 ATM 取现经验。

3. ATM 取现手续费差异较大。
   不同 ATM 和不同发卡银行收费不同；帖子里出现过本地 ATM 1%、10%、固定手续费、以及疑似免本地手续费的 NBU ATM 等不同说法。实际以机器页面和自己银行卡规则为准。

4. Visa / Mastercard 可以作为刷卡和 Yandex Go 绑卡备选。
   但帖子反馈，Visa 刷卡或交通扣款可能加 1.5% 或 2.5% 手续费；Yandex Go 中 Visa / Master 信用卡成功率高于银联卡。

5. 商户刷卡可以用，但不要完全依赖。
   商超、商场、地铁、公交有刷卡经验；但银联有时刷不成，Visa 可用但可能有附加手续费。

6. Yandex Go 最稳妥方案仍是现金或 Visa/Master。
   有帖子尝试通过把地址切到阿拉木图绑定银联/Visa；银联可绑定但扣款可能失败，Visa 在塔什干和撒马尔罕扣款更稳定。

## 二、支付/取款方式对比

### 方式 1：银联卡 ATM 取现

适用场景：
- 旅行者需要现金索姆。
- 没带大量美元或不想频繁换汇。
- 在塔什干、撒马尔罕等城市找到带银联标识的 ATM。

流程：
1. 找带银联标识的 ATM。
2. 插入银联卡。
3. 选择 English。
4. 输入银行卡密码。
5. 选择 withdraw / cash withdrawal。
6. 选择固定金额或 Other amount 手动输入金额。
7. 查看取现金额和手续费。
8. 确认。
9. 按机器提示先取卡，再取钱或先取钱再取卡。
10. 保存小票，核对国内银行扣款。

优点：
- 多篇实测可用。
- 比临时找换汇点更方便。
- 可以按需取现金。
- 部分 ATM 或部分银行卡组合手续费相对可接受。

缺点：
- 手续费不固定。
- 机器限额不一，有的最大单笔 40 万索姆。
- 查询余额可能收手续费。
- 不是所有 ATM 都支持银联。
- 部分 ATM 可能金额选项较小，需要选 Other amount。

注意事项：
- 不要随便查询余额，部分国外 ATM 查询余额也会收费。
- 输入金额时注意不要多输 0。
- 先确认手续费页面再点确认。
- 取现失败时换 ATM 或换银行。

### 方式 2：NBU ATM 银联取现

适用场景：
- 想尝试较低本地手续费的 ATM。
- 在塔什干能找到 NBU ATM。

优点：
- 有帖子称听说 NBU 路边 ATM 取现金免乌国手续费。
- 作者评论确认是银联卡取现方向。

缺点：
- 原帖作者本人说“特地去找，没寻着，在别的 ATM 取完现金才发现隐藏在路边的这个 ATM”，说明位置可能不明显。
- 是否免本地手续费需要现场再确认。

建议：
- 如果路线附近有 NBU ATM，可优先尝试。
- 但不要只押注 NBU；准备其他 ATM 或现金备选。

### 方式 3：Visa / Mastercard ATM 取现

适用场景：
- 没有银联卡。
- 银联 ATM 不好找或取现失败。
- 持 Visa / Mastercard 储蓄卡或信用卡。

流程：
1. 找支持 Visa / Mastercard 的 ATM。
2. 插卡，选择 English。
3. 选择 cash withdrawal。
4. 输入金额。
5. 确认手续费和金额。
6. 取卡取钱。

优点：
- 国际接受度高。
- 在部分 ATM 和 Yandex Go 里比银联更稳定。

缺点：
- 可能有本地 ATM 手续费。
- 发卡银行可能另收境外取现手续费、货币转换费或利息。
- 有帖子提到 Visa 取现/刷卡可能 1.5% 手续费。

### 方式 4：银行卡刷卡消费

适用场景：
- 商场、超市、部分餐厅、交通等支持刷卡场景。

经验总结：
- 有帖子实测地铁和公交可以刷 Visa，车票 1700 索姆，手续费约 1.5%。
- 超市和商场多次刷 Visa，帖子称会加 1.5% 或 2.5% 手续费出账。
- 机器上即使有银联标志，也可能刷不成功。
- 作者建议：能刷银联时优先刷银联，因为其经验中银联没有额外收手续费。

优点：
- 不用携带大量现金。
- 商场/超市等场景方便。

缺点：
- 手续费不透明。
- 银联标志不代表一定能刷成功。
- 小店、市场、出租车等场景仍可能只收现金。

### 方式 5：现金索姆

适用场景：
- 市场、巴扎、小店、出租车、临时购票、小额消费、手机卡办理等。

优点：
- 接受度最高。
- 适合小额消费。
- 在网络不好、刷卡失败、Yandex Go 绑卡失败时兜底。

缺点：
- 需要保管现金。
- ATM 单笔限额或换汇时需要排队。
- 大额钞找零可能麻烦。

建议：
- 入境后尽早取一笔现金。
- 每次不必取太多，但要保证至少有打车、吃饭、买卡、应急的钱。

### 方式 6：Yandex Go 绑卡支付

适用场景：
- 打车、外卖等 Yandex Go 生态支付。

流程：
1. 下载 Yandex Go 并登录。
2. 进入左上角设置。
3. 将当前位置/地址切到阿拉木图。
4. 再进入银行卡添加页面。
5. 尝试绑定 Visa / Mastercard / 银联卡。
6. 如果绑不上，打开银行卡的境外支付、小额免密等功能后再试。
7. 绑定后回到塔什干/撒马尔罕使用。

优点：
- 成功后打车不用准备现金。
- Visa / Mastercard 信用卡成功率较高。

缺点：
- 地址在中国或乌兹别克斯坦时可能无法成功绑定。
- 银联卡即使能绑定，扣款也可能失败。
- 帖子中明确说：银联卡在撒马尔罕打车扣款失败；Visa 在塔什干和撒马尔罕扣款成功。
- Yandex Go 不绑卡时默认现金付款，司机找零可能不完全，比如 500 或 1000 索姆不找。

建议：
- 最推荐绑定 Visa / Mastercard。
- 银联卡可试，但不要依赖。
- 无论是否绑卡，都准备现金索姆。

## 三、推荐组合

### 短期旅行推荐

1. 银联储蓄卡一张：用于 ATM 取现。
2. Visa 或 Mastercard 一张：用于刷卡、Yandex Go 绑卡、备用取现。
3. 少量美元或人民币现金：作为无法取现时的备用换汇。
4. 到达后取一笔索姆现金：覆盖打车、吃饭、电话卡、小额消费。
5. Yandex Go 优先绑 Visa / Mastercard；绑不上就现金付款。

### 更稳妥做法

- 第一优先：现金索姆兜底。
- 第二优先：银联卡 ATM 取现。
- 第三优先：Visa / Mastercard 刷卡和 Yandex Go。
- 不建议依赖微信/支付宝换汇或私人换汇。

## 六、最终建议

如果只带一种支付方式，不够稳。
建议组合如下：

1. 银联储蓄卡：主力 ATM 取现。
2. Visa / Mastercard：刷卡和 Yandex Go 备用。
3. 现金索姆：所有小额场景和打车兜底。
4. 少量美元现金：无法 ATM 取现时备用。
5. 不依赖微信/支付宝换汇。
